Project Objective & Analytical Methods

This page combines what we are trying to learn (objectives) with how we analyze the data (methods), so presentation navigation stays consistent across the site.

Project Objective

This project studies professional League of Legends career trajectories to understand how long careers typically last, how players transition from Tier 2 to Tier 1 leagues, and which structural factors are associated with longer career longevity.

Main goals

  1. Estimate the typical career length distribution for pro players from Seasons 6-15.
  2. Measure Tier 2 to Tier 1 transition rates and timing.
  3. Compare career outcomes across major regions (LCK, LPL, LEC, LCS).
  4. Provide an interpretable prediction service for expected career length.

Research questions

  • What is the typical length of a professional LoL career?
  • How often do Tier 2 debuts reach Tier 1 leagues?
  • Which regions show stronger long-career retention patterns?
  • How do role, debut era, and league context change expected longevity?

Expected impact

Why this matters

  • Players: better expectations for career planning.
  • Teams: data-informed scouting and roster development.
  • Leagues: clearer understanding of development pipeline outcomes.
  • Researchers: reproducible baseline for esports career studies.

Pipeline Overview

  1. Data collection. We collected publicly available esports records using scripts in scrapers/, including roster and league history needed to reconstruct career timelines.
  2. Data cleaning. We standardized player identifiers, league strings, roles, and season ranges so different sources can be joined consistently into analysis-ready tables.
  3. Feature engineering. We derived career-level outcomes such as career length, debut era indicators, and tier transition flags from the cleaned timeline data.
  4. Exploratory analysis. We summarized distributions, missingness patterns, and cohort structure to guide modeling choices and to sanity-check coverage before hypothesis testing.
  5. Hypothesis testing and cohort analysis. We compared groups using nonparametric tests where appropriate and built Tier 2 cohort summaries to quantify promotion outcomes and timing.
  6. Modeling and deployment. We trained survival and count models for career longevity, packaged them for inference, and published results through the website with interactive charts and a live predictor API.

Methods Used

Method Purpose Where used
EDA (distribution + cohort summaries) Understand baseline structure and data quality pages/eda.html (supporting) and generated figures
Kruskal-Wallis + Mann-Whitney U Compare career lengths across leagues Summarized on pages/findings.html
Tier transition cohort analysis Measure Tier 2 to Tier 1 promotion rates/timing pages/hypothesis2.html (interactive) and pages/findings.html
Cox Proportional Hazards Estimate survival probability over seasons Inference API and pages/predictor.html
Negative Binomial Regression Predict expected career length (count outcome) Inference API and pages/predictor.html

Model Inputs and Outputs

Inference input features

  • League: LCK, LPL, LEC, LCS
  • Role: Top, Jungle, Mid, Bot, Support
  • Debut season: integer season index (S6-S15)

Inference outputs

  • Predicted expected career length (seasons)
  • Season-by-season survival probabilities
  • Median survival season (first point below 50%)
  • Confidence label and human-readable interpretation
